# ortCI
Basic ort flow 

## Usage
```

$ bash .ort.sh
=============== Please choose run with below options ===============
1. bash .ort.sh scan  -> to initiate new scan
2. bash .ort.sh debug -> to get the report for debug
3. bash .ort.sh bom   -> to get bill of materials

```
